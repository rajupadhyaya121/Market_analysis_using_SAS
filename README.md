# Marketing Campaign Analysis using SAS

## Objective
Analyze a bank telemarketing dataset to identify customer segments with higher conversion rates.

## Dataset
Bank Marketing Dataset (Kaggle / UCI Machine Learning Repository)

## Tools & Technologies
- Base SAS (SAS Studio)
- PROC SQL
- PROC MEANS
- PROC FREQ
- PROC SGPLOT

## Key Analysis
- Overall campaign response analysis
- Customer segmentation by age and job
- Loan and balance impact on conversion
- Summary statistics for financial profiling

## Outcome
Identified high-conversion customer segments to support targeted marketing strategies.
